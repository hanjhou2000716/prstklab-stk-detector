## 使用者驗收步驟

完成以下四項，即可視為研究版 Beta 可日常使用：

- [ ] Telegram 可收到一則不超過 30 字的快報，且按鈕能打開儀表板。
- [ ] 儀表板顯示代表標的、新聞／風控、研究卡片與資料狀態。
- [ ] 手動執行 **Unified Taiwan-US research report** 後，Actions 顯示成功，儀表板的「台美全市場研究摘要」有資料或明確顯示無候選／來源缺漏。
- [ ] 在該次 Actions 下載 Artifact，能看到 CSV 與 JSON 報表。

## Beta 的範圍與尚未包含的項目

已包含：公開資料快報、GitHub Pages、台美研究掃描、風險與新聞摘要、歷史研究回測、研究配置草案、再平衡檢視與資料健康度。

不包含：券商串接、自動下單、持倉讀取、目標價、買賣建議、保證績效、個人化投資推薦。

## 驗收不通過時

先在 Actions 開啟失敗的工作流程，查看紅色步驟名稱。若是公開資料失敗，等一段時間後重跑即可；若是 Telegram 或 Pages 相關失敗，保留該次 Actions 網址並回到本對話，我會根據紀錄處理。
